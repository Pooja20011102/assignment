Welcome to the Type, Class, and ID Selector Demo

This paragraph is styled using a class selector. Class selectors are useful when you want to apply the same style to multiple elements. For example, this pink box could be reused for different sections like tips, notes, or warnings.

This div is styled using an ID selector, which should be unique within the page. You can use an ID for highlighting one specific area like a featured product, a main announcement, or a special banner section.

Type selectors, class selectors, and ID selectors together form the foundation of CSS styling. They help create a clear hierarchy and structure in your web design.
