Content Box

Its total rendered width will be 200px (content) + 20px (left padding) + 20px (right padding) + 5px (left border) + 5px (right border) = 250px. Similarly, its total rendered height will be 100px + 20px + 20px + 5px + 5px = 150px



Border Box

The content area's width will be 200px - (20px + 20px + 5px + 5px) = 150px, and its height will be 100px - (20px + 20px + 5px + 5px) = 50px.
